# DevTeam - Software Development Team Management SaaS

A lightweight, single-page application for managing development teams, projects, and tasks. Built with vanilla HTML, CSS, and JavaScript.

## Features
- **Dashboard**: Overview of project status, pending tasks, and team stats.
- **Projects**: Create and track projects with progress bars.
- **Tasks**: Assign tasks to team members with priority levels.
- **Team**: Manage team members and roles.
- **Data Persistence**: All data is saved automatically to your browser's Local Storage.

## How to Run
1. Simply double-click `index.html` to open the application in your web browser.
2. No server or installation required.

## Technologies
- HTML5
- CSS3 (Custom properties, Flexbox/Grid)
- JavaScript (ES6+)
- Ionicons (User Interface Icons)
- Inter Font (Typography)
